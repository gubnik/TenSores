Detailed Description

template<typename T, std::size_t Order, Allocator A = std::allocator<T>>
class TenSore::Tensor< T, Order, A >::Iterator

Iterator class of Tensor.

The Iterator class represents an iterator of Tensor. It contains an index to which it points, and provides necessary operations for STL integration, as well as thread-safety.
